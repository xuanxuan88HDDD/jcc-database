# 金铲铲之战数据库 (JCC Database)

金铲铲之战数据库是一个专门用于展示金铲铲之战游戏数据的Web应用，包括英雄、物品、羁绊等信息。该项目能够自动扫描jcc目录中的所有赛季数据，并提供动态展示和检索功能。

## 功能特性

- **多赛季数据支持** - 支持jcc目录中所有赛季和模式的游戏数据
- **英雄图鉴** - 展示所有英雄的详细信息，包括技能、羁绊、属性等
- **物品图鉴** - 展示所有装备的属性和合成路径
- **赛季档案** - 展示不同赛季的特色内容和更新记录
- **智能搜索** - 支持按名称、赛季、类型等多种条件搜索
- **响应式设计** - 适配桌面、平板和手机设备
- **动态数据加载** - 自动扫描jcc目录并加载最新数据

## 技术栈

- **前端框架**: React 18
- **路由**: React Router
- **UI样式**: Tailwind CSS
- **状态管理**: React Hooks
- **构建工具**: Vite
- **图标库**: Lucide React
- **数据格式**: JSONP (从jcc目录加载)
