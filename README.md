# 金铲铲之战数据库 (JCC Database)

金铲铲之战是一款流行的策略战棋类手机游戏，本项目是一个用于展示游戏中各种数据的Web应用，包括英雄、物品、羁绊等信息。

## 项目简介

本项目旨在为金铲铲之战玩家提供一个全面的游戏数据库，帮助玩家更好地了解游戏中的各种元素，包括：

- **英雄图鉴** - 展示所有英雄的详细信息，包括技能、羁绊、属性等
- **物品图鉴** - 展示所有装备的属性和合成路径
- **赛季信息** - 展示不同赛季的特色内容

项目采用现代Web技术栈构建，具有响应式设计，可在各种设备上流畅使用。

## 技术栈

- **前端框架**: React 18
- **样式框架**: Tailwind CSS
- **构建工具**: Vite
- **图标库**: Lucide React
- **开发语言**: JavaScript

## 功能特点

- **动态数据加载** - 自动从jcc目录加载所有赛季数据
- **智能搜索** - 支持按名称搜索英雄和物品
- **多维度筛选** - 按赛季、羁绊、费用等多种条件筛选
- **响应式设计** - 适配桌面端和移动端
- **实时预览** - 点击即可查看详细信息

## 安装与运行

1. 克隆项目
   ```bash
   git clone <repository-url>
   ```

2. 安装依赖
   ```bash
   cd jcc-database
   npm install
   ```

3. 启动开发服务器
   ```bash
   npm run dev
   ```

4. 访问应用
   打开浏览器访问 `http://localhost:5173`

src/
├── components/           # 可复用UI组件
│   ├── HeroCard.jsx     # 英雄卡片组件

└── App.jsx              # 主应用组件

## 数据来源

项目数据存储在 `jcc/` 目录下，按不同赛季和模式分组：

- `冠军乱斗_202501_S16_15.15.4/` - 冠军乱斗模式 S16 赛季
- `双城传说2_13_S14_14.13.5/` - 双城传说模式 S14 赛季
- `天下无双格斗大会_15_S16_15.15.4/` - 天下无双格斗大会模式 S16 赛季
- 更多赛季...

数据格式为JSONP，包含英雄、物品、羁绊等游戏元素的详细信息。

## 特色功能
